Tween

Kind of class:class
Inherits from:MovieClip
Classpath:gfx.motion.Tween
File last modified:Monday, 05 April 2010, 10:09:52
The Tween class adds tween methods tweenTo() and tweenFrom() to all MovieClips when the Tween.init() method is called.

Summary


Class methods
  • linearEase
    • The default ease equation.
  • init
    • Initialize the tween functions in the MovieClip class.
Instance methods
  • tweenTo
    • Tween the properties of a MovieClip specified in the props object.
  • tweenFrom
    • Tween the properties of a MovieClip from the properties specified in the props object to the current properties.
  • tweenEnd
    • End a tween immediately.

init

static function init (
) : Boolean

Initialize the tween functions in the MovieClip class. Once initialized, any MovieClip can make use of the Tween methods.
Returns:
  • Whether Tween was initialized in the current call. If Tween was already initialized, it will return false.
